# Object Types : 
    - Numbers : 1234 , 2.13 , 3+2j , 0b111 , Decimal(), Fraction() 
    - String : 'spam' , "spa" , etc ...
    - List : [1,2,3] , ["spa","pass"] , [[],[] ...]
    - Tuple : (1, 'spa', 4,"U") , namedtuple
    - Dictionary : {'food':'spam', "tast":""}  key : value

    - Set : set('abc'),{'a','b'}
    - File : open('eggs.txt')
    - Boolean : True , False
    - None : None

    - Functions , modules , classes
